# Workflow: Owner Onboarding

## Steps

1. Land on `/` → CTA Sign up
2. `/signup` → submit name/email/password
3. Session created → `/` owner dashboard
4. Empty pets → CTA Add pet
5. `/pets/new` → save pet
6. Optional: browse `/clinics`

## Acceptance

- New owner reaches dashboard with ≥0 pets without console errors
- Validation errors do not clear the whole form unexpectedly
