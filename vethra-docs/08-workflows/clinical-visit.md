# Workflow: Clinical Visit (Demo Path)

1. Clinic confirms appointment
2. Vet opens patient from appointment or `/patients/:id`
3. Reviews timeline + allergies
4. Adds diagnosis (+ notes, treatment plan)
5. Adds prescription
6. Creates medication/follow-up reminder (+ optional treatment task)
7. Owner sees Rx/reminders/tasks; marks task completed
8. Timeline shows full story on next open

## Demo tip

Pre-seed a pet with one prior timeline item so “history” is visible before adding new diagnosis.
