- [ ] CORS locked to real origin
- [ ] Mongo IP allowlist / auth
- [ ] Email credentials valid
- [ ] Seed not running accidentally in prod
- [ ] Error stacks hidden
- [ ] HTTPS only
- [ ] Backup/export plan for demo day
