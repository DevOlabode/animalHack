# Goals

## Hackathon goals

1. Prove owners can find a clinic and book with a clear reason.
2. Prove clinics can triage appointments (accept/decline) with pet context.
3. Demonstrate a credible medical timeline after a visit.
4. Ship a demo-ready, responsive web app.
5. Leave architecture ready for SaaS growth (no throwaway schemas).

## Product goals (post-hackathon)

1. Become the default shared record between owners and clinics.
2. Improve appointment show-up and treatment compliance.
3. Monetize via clinic subscriptions (owners free initially).
4. Expand discovery (reviews, geo search) and engagement (articles, later social).

## Non-goals (hackathon)

- Perfect design system parity with a design agency.
- Full HIPAA-equivalent compliance program (still apply security hygiene).
- Building every vision feature from the mind map.
