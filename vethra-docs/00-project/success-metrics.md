# Success Metrics

## Hackathon / demo success

Binary: the demo flow in `12-hackathon/demo-script.md` completes without critical bugs.

Checklist:

- [ ] Owner registers and adds ≥1 pet
- [ ] Clinic registers with profile
- [ ] Owner books appointment with reason
- [ ] Clinic sees pet + reason and confirms
- [ ] Vet records diagnosis + prescription
- [ ] Reminder exists; owner can update a treatment task
- [ ] Timeline shows the visit artifacts

## Product metrics (post-launch)

| Metric | Definition | Target (early) |
|--------|------------|----------------|
| Booking completion rate | Booked / started booking | > 60% |
| Confirmation latency | Time pending → confirmed/declined | < 24h median |
| Timeline adoption | % completed appts with diagnosis | > 70% |
| Reminder open/action | Tasks marked completed | Trend up |
| Clinic activation | Clinics with ≥5 confirmed appts/month | Growth |

## Qualitative

Judges/users understand the value in under 60 seconds of demo.
