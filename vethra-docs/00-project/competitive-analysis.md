# Competitive Analysis

## Category landscape

| Type | Examples (illustrative) | Gap |
|------|-------------------------|-----|
| Owner-only pet apps | Pet journals, vaccine trackers | No clinic workflow |
| Clinic PMS | Desktop practice software | Weak owner experience |
| Marketplace booking | Generic appointment tools | No shared medical timeline |
| Telehealth startups | Video-first vet apps | Heavy infra; not MVP |

## Vethra differentiation

1. **Shared timeline** — owner + clinic same source of truth
2. **Reason-first booking** — clinic sees problem before acceptance
3. **Post-visit loop** — Rx + reminders + compliance tasks
4. **Clinic-paid SaaS path** — clear monetization later

## Positioning statement

For pet owners and small clinics who need coordinated care, Vethra is a shared pet healthcare OS that connects booking, records, and follow-up—unlike owner-only journals or clinic-only PMS tools.
