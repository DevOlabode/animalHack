# Vision

**Phase:** MVP (Hackathon) (vision spans all phases)

## North star

Every pet has a **digital medical passport**. Every veterinarian who treats that pet contributes to one timeline. Every owner can manage care without paperwork.

## Long-term product (vision)

### Pet owners

| Capability | Phase |
|------------|-------|
| Register multiple pets with photos & details | MVP |
| Browse clinics / private vets; view profile (hours, contact, values, doctors) | MVP (basic) → Phase 2 (rich) |
| Book appointments with reason / problem description | MVP |
| Cancel appointments with reason | MVP |
| Medical timeline, prescriptions, reminders, treatment tasks | MVP |
| Reviews of clinics | Phase 2 |
| Social feed: fun moments with pets | Future |
| Messaging with clinics | Future |
| Vaccination alerts, history export | Phase 2 |

### Clinics / private vets

| Capability | Phase |
|------------|-------|
| Clinic profile & registration | MVP |
| Booking notifications (email) | MVP |
| See pet + reason before appointment | MVP |
| Calendar: pending / confirmed / declined / completed | MVP |
| Diagnosis, prescriptions, reminders | MVP |
| Patient search | MVP |
| Publish articles | Future |
| Staff accounts, analytics | Phase 2–3 |
| Telemedicine | Future |

## Product positioning

Not just “pet record storage” — a **pet care management system** / veterinary care OS bridging owner ↔ clinic ↔ pet.

## What we are not (hackathon)

Payments, insurance claims, AI diagnosis, native mobile apps, prescription fulfillment, complex marketplace rankings.
