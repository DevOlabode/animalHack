# Solution

## Product

**Vethra** — web app (React + Express + MongoDB) connecting pet owners and veterinary clinics.

## Core value loops

### Loop A — Discover & book (hackathon hero)

1. Owner signs up and registers pets.
2. Owner browses clinics and opens a clinic profile.
3. Owner books a slot and explains the reason.
4. Clinic is notified, sees pet + reason, accepts or declines.
5. Owner sees status; visit is on the clinic calendar.

### Loop B — Care & continuity (MVP depth)

1. Vet opens patient timeline.
2. Vet records diagnosis and prescription.
3. Vet schedules reminders / care tasks.
4. Owner receives emails and marks tasks done.
5. Next visit starts with full context.

## Why this beats a simple notes app

Vethra **participates in care**: booking, status workflow, reminders, and compliance—not only storage.
