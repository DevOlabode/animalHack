# Accessibility

Target: WCAG 2.1 AA where practical for MVP forms and navigation.

## Checklist

- [ ] All inputs have labels
- [ ] Error messages linked via `aria-describedby` or adjacent text
- [ ] Focus order follows visual order
- [ ] Modals trap focus and restore on close
- [ ] Color contrast AA for text/buttons
- [ ] Status badges include text
- [ ] Images have alt (pet name)
- [ ] Buttons announce loading state (`aria-busy`)

## Manual test

Keyboard-only pass through booking + accept flows.
