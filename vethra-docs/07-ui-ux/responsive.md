# Responsive Design

## Breakpoints (suggested)

| Name | Width |
|------|-------|
| sm | 640 |
| md | 768 |
| lg | 1024 |

## Requirements

- All MVP pages usable on mobile browser
- Nav collapses appropriately
- Booking form single column on small screens
- No horizontal scroll on 360px width

Native apps are out of scope.
