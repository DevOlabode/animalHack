# Typography

## MVP

Use the project’s existing font stack. If introducing fonts:

- **Display/Brand:** distinctive but readable
- **UI/Body:** highly readable sans for forms and tables

## Scale (suggested)

| Token | Size | Use |
|-------|------|-----|
| display | 2–2.5rem | Landing brand/hero |
| h1 | 1.75rem | Page titles |
| h2 | 1.35rem | Section |
| body | 1rem | Default |
| small | 0.875rem | Meta, timestamps |

Line length for reading blocks ~60–75ch.
