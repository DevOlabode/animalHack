# Demo Script (≈5 minutes)

## Setup

Two browsers/profiles: Owner and Clinic. Seed optional clinic.

## Script

1. **Hook (20s):** “Records and phone tag fail pets. Vethra is a shared care OS.”
2. **Clinic exists** — show clinic profile (hours, contact).
3. **Owner signup / login** — add pet with photo + allergies.
4. **Book** — choose clinic, slot, paste a vivid reason (“coughing 3 days, off food”).
5. **Switch to clinic** — show notification/email or pending queue; highlight pet + reason.
6. **Accept** — show confirmed on calendar/board.
7. **Visit** — open patient timeline; add diagnosis + Rx + reminder.
8. **Owner** — show Rx/reminder; mark treatment task completed.
9. **Close** — “That’s find → book → care → follow-up in one product. Next: reviews, staff accounts, mobile.”
