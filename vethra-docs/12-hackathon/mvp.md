# Hackathon MVP

## One problem

**Make it easy for pet owners to find a veterinarian and book appointments online — with enough clinical context to continue care.**

## Must build

1. Auth (owner + clinic)
2. Pet profiles (multi-pet)
3. Clinic browse + profile
4. Book with **reason**
5. Cancel with **reason**
6. Clinic notifications (email) + pending queue
7. Accept / decline / complete
8. Calendar or status-filtered appointment views
9. Medical timeline + diagnosis + prescription
10. Reminders + treatment task status
11. Patient search
12. Responsive UI

## Must not build (mind-map deferrals)

- Social pet moments feed
- Clinic articles
- Reviews / star ratings
- Full team directory with photos (unless trivial static fields)
- Geo maps
- Payments
- Telemedicine
- Native apps

## Definition of done

Demo script completes cold on a clean seed in < 5 minutes.
